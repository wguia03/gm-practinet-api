# PracticasYa! API

- API RESTful para un portal de empleo dirigido a practicantes profesionales y preprofesionales a nivel nacional
- Autenticación y autorización con JWT
- Despliegue del servicio y la base de datos en Render

## Tech Stack

- Java 21
- Spring Framework
- Spring Boot
- Spring Data JPA
- Hibernate
- Spring Security
- PostgreSQL
- Docker
- Swagger
